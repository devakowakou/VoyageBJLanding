# **App Name**: VoyageBJ Landing

## Core Features:

- Hero Section Search: Implement a search form in the hero section that allows users to input departure city, arrival city, and date to find available routes.
- Company Partnership Section: Display information for transport companies interested in joining the VoyageBJ platform, detailing benefits such as increased visibility and analytics tools.
- User Testimonials: Showcase testimonials from satisfied users, including photos, star ratings, and quotes about their positive experiences with VoyageBJ.
- Statistics Display: Present key statistics like the number of available routes, partner companies, and satisfied travelers using animated counters to highlight VoyageBJ's impact.
- Team Introduction: Briefly introduce the team and their vision, with a call to action to join the team via a link to Discord or email.
- Dark/Light Mode Toggle: Add a toggle allowing users to switch between dark and light mode for comfortable viewing in different environments.

## Style Guidelines:

- Primary color: Royal Blue (#2563EB) for trust and stability.
- Secondary color: Emerald Green (#10B981) for a sense of growth and reliability.
- Background color: Light Gray (#F3F4F6) to maintain a clean and modern interface.
- Headline font: 'Poppins' sans-serif for clear, modern headlines.
- Body font: 'Inter' sans-serif for readable, simple body text.
- Use vector icons and a stylized map of Benin to enhance visual appeal and user understanding.
- Employ subtle animations like fade-ins and parallax scrolling to create a fluid and engaging user experience.
